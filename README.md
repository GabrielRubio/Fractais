# 1. Fractal Sierpinski Triangle

Compile:
```
$ javac SierpinskiTriangle.java
```

Fractal with 4 interactions:
```
$ java SierpinskiTriangle 4 
```

```
$ java SierpinskiTriangle 6 
```
